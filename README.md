# 					Инструкция по запуску приложения ВГУВТ.Расписание

## 1. Установка драйверов.

Обычно драйвера уже включены в пакеты операционной системы вашего компьютера либо установятся автоматически при подключении, но если возникают проблемы с обнаружением устройства, то необходимо скачать необходимые драйвера . 

## 2. Включение отладки по usb.

После установки драйверов, необходимо включить отладку по usb, для этого необходимо зайти в настройки, информация о телефоне и нажать несколько раз на строчку "Номер сборки". После чего появится уведомление о том, что включен режим разработчика. Теперь в меню дополнительных настроек либо в меню настроек появилось новое меню - "Возможности разработчика". В меню возможностей разработчика листаем до раздела "отладка" и нажимаем "отладка по usb".

## 3. Подключение смартфона к компьютеру.

После включении отладки по usb, мы подключаем наш смартфон к компьютеру с помощью usb кабеля, который идет при покупки вместе со смартфоном. В момент подключения, на экране смартфона появится уведомление о выборе режима подключения. Выбираем подключение в режиме передачи файлов. Затем появится уведомление о разрешении отладки по usb данному компьютеру. Нажимаем разрешить.

## 4. Установка Android studio.

После настройки нашего смартфона необходимо установить Android studio на компьютер. 

### 1. Установка Android studio в linux.

Для установки Android studio в linux необходимо зайти на [оффициальный сайт](https://developer.android.com/studio) android studio .

Нажать кнопку скачать Android studio.

После загрузки файла, необходимо его распаковать, для этого нужно открыть терминал, перейти в папку куда был скачан файл и распаковать его командой:

sudo unzip "название файла" -d папка куда вы хотите распаковать файлы android studio.

Далее, для запуска android studio необходимо перейти в терминале в распакованную папку android-studio далее в папку bin и запустить studio.sh.



Теперь при запуске файла studio.sh из папки bin будет открываться уже установленная Android studio.

Для удобства можно создать ярлык быстрого запуска, для этого переходим в папку ~/.local/share/applications/

И создаем файл androidstudio.desktop

В который помещаем такое содержание: 



    [Desktop Entry]
    Version=1.0
    Type=Application
    Name=Android Studio
    Exec="/home/imaginator/Документы/android-studio/bin/studio.sh" %f
    Icon=/home/imaginator/Документы/android-studio/bin/studio.png
    Categories=Development;IDE;
    Terminal=false
    StartupNotify=true
    StartupWMClass=android-studio

где в пункте Exec и Icon указываем путь к распакованным файлам studio.sh и studio.png в папке bin.

### Установка Android studio в windows.

Переходим на [оффициальный сайт](https://developer.android.com/studio?hl=ru) android studio и нажимаем скачать.

После загрузки инсталятора android studio запускаем его.

Появляется окно с выбором места установки программы, выбираем нужную директорию.
Затем появляется окно с выбором типа настройки для Android Studio. Выбираем стандартную версию.

Затем пойдет установка программы. После окончания установки нажимаем Finish.

## Синхронизация, компиляция и запуск проекта в Android studio.

Запускаем Android studio.

После запуска android studio появится окно, в котором нас спросят нужно ли импортировать конфигурацию или создать новую. 

Выбираем не импортировать конфигурацию и нажимаем кнопку ОК.

Далее появится окно сообщающее, что нужно установить необходимые файлы. Затем нажимаем Next.

Затем появится окном с выбором стандартной или пользовательской установки. Нажимаем Standart. Затем Next.

Далее появится окно с ознакомлением загружаемых компонентов. Нажимаем  Next.

Затем появится окно,сообщающее, что установка завершена. Нажимаем Finish.

После настройки Android studio нажимаем кнопку открыть существующий проект. Открывается окно с выбором проекта. Ищем и выбираем папку с проектом . Затем открывается окно Android studio. Ждем пока наш проект синхронизируется. Нажимаем кнопку молотка для компиляции проекта. После успешной компиляции, разблокируем телефон, в Android studio нажимаем кнопку play справа от кнопки молотка. Ждем пока приложение запуститься.
